# scoutmini_description

This package contains the Scout Mini robot description, including URDF/Xacro,
mesh, RViz, and related configuration assets.

## Mesh Attribution

The following mesh files were copied from the AgileX Robotics `scout_ros`
repository:

- `meshes/scout_mini_base_link.dae`
- `meshes/wheel.dae`

Source:
https://github.com/agilexrobotics/scout_ros/scout_description/meshes
